### Этап 2: Frontend - Визуальный лоск и киоск-режим

#### ✅ 2.1 Динамические аватарки сотрудников
- **Файл:** `frontend/src/views/Staff.vue`
- **Проблема:** Пустые карточки без фотографий
- **Решение:** Интеграция с UI Avatars API
- **Результат:** Красивые аватарки с инициалами для всех 109 сотрудников

**Реализация:**
```javascript
const getAvatarUrl = (person) => {
  if (person.photo_url) return person.photo_url
  
  const name = encodeURIComponent(person.full_name)
  return `https://ui-avatars.com/api/?name=${name}&background=random&color=fff&size=200&bold=true&font-size=0.4`
}
```

**Визуальные улучшения:**
- Круглые аватарки 120x120px
- Случайные цвета фона
- Анимация при наведении (scale 1.05)
- Белая рамка с тенью

#### ✅ 2.2 Таймер бездействия для киоска
- **Файл:** `frontend/src/App.vue`
- **Проблема:** Пользователи оставляют киоск на внутренних страницах
- **Решение:** Автовозврат на главную после 3 минут бездействия
- **Результат:** Киоск всегда готов для нового посетителя

**Отслеживаемые события:**
- `mousedown` - нажатие мыши
- `mousemove` - движение мыши
- `keypress` - нажатие клавиш
- `scroll` - прокрутка
- `touchstart` - касание экрана
- `click` - клики

**Логика:**
- Любая активность сбрасывает таймер
- Через 3 минуты бездействия → переход на главную
- Если уже на главной → ничего не происходит
- Корректная очистка слушателей при размонтировании

## 🚀 Как запустить проект
