# Быстрый старт проекта

## ✅ Что уже сделано:

1. **База данных создана** - SQLite с таблицами
2. **Данные загружены**:
   - 109 сотрудников из DOCX
   - 73 записи расписания из PDF
3. **Backend запущен** - FastAPI на http://localhost:8000
4. **API работает** - можно тестировать endpoints

## 🚀 Запуск Frontend

Теперь нужно запустить фронтенд:

```bash
cd frontend
npm install
npm run dev
```

После этого откройте http://localhost:5173

## 🧪 Тестирование API

```bash
# Все сотрудники
curl http://localhost:8000/api/v1/staff/

# Поиск сотрудника
curl "http://localhost:8000/api/v1/staff/search?query=Алексеев"

# Расписание группы
curl "http://localhost:8000/api/v1/schedule/group/9ИС-1.25"

# Документация API
http://localhost:8000/docs
```
