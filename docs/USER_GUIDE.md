# Руководство пользователя

## Интерактивный информационный стенд ККРИТ

Стенд предназначен для быстрого доступа к расписанию, сотрудникам, новостям и навигации по корпусу колледжа. Устанавливается на сенсорный киоск или доступен через браузер.

---

## Главная страница

При открытии стенда отображается главная страница с:
- **Текущее время и дата**
- **Ближайшие занятия** — расписание на сегодня для выбранной группы
- **Последние новости** — 3 свежие новости с сайта колледжа
- **Быстрые переходы** — кнопки для перехода к основным разделам

---

## 📅 Расписание

### Поиск расписания

1. Нажмите на раздел **«Расписание»** или кнопку на главной
2. Выберите тип поиска:
   - **По группе** — введите название группы (например, `9РВП-1.25`)
   - **По преподавателю** — введите фамилию (например, `Курбанова`)
   - **По аудитории** — введите номер кабинета (например, `201`)
3. Начните вводить — появится выпадающий список с подсказками
4. Выберите нужный вариант из списка

### Выбор даты

- Под строкой поиска расположена **карусель дат**
- Свайпайте влево/вправо или используйте колёсико мыши для прокрутки
- Нажмите на нужную дату для фильтрации расписания
- Кнопка 📅 открывает календарь для быстрого перехода к дате

### Типы занятий

Каждое занятие отмечено цветным значком:

| Значок | Тип занятия |
|--------|------------|
| 🟣 Лек | Лекция |
| 🔵 Практ | Практическое занятие |
| 🟢 Лаб | Лабораторная работа |
| 🟠 Дист | Дистанционное занятие |
| КЧ | Классный час (понедельник, 1-я пара) |

### Время занятий

Время автоматически определяется по дню недели:
- **Понедельник**: 1-я пара — КЧ (08:00–08:55), затем пары с 09:00
- **Вторник–Суббота**: 1-я пара с 08:00

## 👥 Сотрудники

1. Откройте раздел **«Сотрудники»**
2. Используйте строку поиска для поиска по ФИО или должности
3. Нажмите на карточку сотрудника для подробной информации

На карточке отображаются:
- ФИО и инициалы (аватарка генерируется автоматически)
- Должность
- Номер кабинета
- Контакты (email, телефон — если указаны)

---

## 🏫 Карта

1. Откройте раздел **«Карта»**
2. Выберите этаж (2 или 3)
3- Используйте кнопки зума для масштабирования
4. Нажмите на кабинет для просмотра информации

---

## 📰 Новости

Новости отображаются на главной странице (3 последние) и автоматически обновляются каждые 6 часов с сайта колледжа.

## Подсказки

- **Свайп** по карусели дат — быстрый переход между днями
- **Нажатие на пару** — разворачивает подробности занятия
- **QR-код** на расписании — можно отсканировать для просмотра на телефоне
- Стенд автоматически возвращается на главную страницу после 5 минут бездействия
