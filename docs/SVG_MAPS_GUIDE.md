# Как создать SVG карты этажей для навигации

## Вариант 1: Figma (Рекомендуется)

**Самый простой способ:**

1. **Зарегистрируйтесь на Figma** (бесплатно): https://figma.com
2. **Создайте новый файл**
3. **Нарисуйте план этажа:**
   - Используйте прямоугольники (R) для кабинетов
   - Добавьте текст (T) с номерами кабинетов
   - Используйте линии для коридоров
   - Разные цвета для разных типов помещений

4. **Экспорт в SVG:**
   - Выделите все элементы
   - Правая кнопка → Export → SVG
   - Сохраните как `floor1.svg`, `floor2.svg` и т.д.

5. **Поместите файлы:**
   ```
   diplom/data/maps/floor1.svg
   diplom/data/maps/floor2.svg
   diplom/data/maps/floor3.svg
   ```

## Вариант 2: Inkscape (Бесплатный редактор)

1. **Скачайте Inkscape**: https://inkscape.org/
2. **Создайте новый документ**
3. **Нарисуйте план:**
   - Инструмент "Прямоугольник" для кабинетов
   - Инструмент "Текст" для номеров
   - Инструмент "Линия" для коридоров

4. **Сохраните как SVG:**
   - File → Save As → Plain SVG

## Вариант 3: Онлайн редактор SVG

**SVG-Edit** (не требует установки): https://svg-edit.github.io/svgedit/

1. Откройте редактор
2. Нарисуйте план этажа
3. File → Save Image → SVG

## Структура SVG файла

Пример простого плана этажа:

```svg
<svg width="800" height="600" xmlns="http://www.w3.org/2000/svg">
  <!-- Коридор -->
  <rect x="50" y="250" width="700" height="100" fill="#ecf0f1" stroke="#95a5a6" stroke-width="2"/>
  
  <!-- Кабинет 305 -->
  <rect id="room-305" x="100" y="100" width="150" height="100" fill="#3498db" stroke="#2c3e50" stroke-width="2"/>
  <text x="175" y="155" text-anchor="middle" fill="white" font-size="20">305</text>
  
  <!-- Кабинет 306 -->
  <rect id="room-306" x="300" y="100" width="150" height="100" fill="#3498db" stroke="#2c3e50" stroke-width="2"/>
  <text x="375" y="155" text-anchor="middle" fill="white" font-size="20">306</text>
  
  <!-- Лестница -->
  <rect x="600" y="200" width="100" height="200" fill="#e74c3c" stroke="#c0392b" stroke-width="2"/>
  <text x="650" y="305" text-anchor="middle" fill="white" font-size="16">Лестница</text>
</svg>
```

## Что нужно отметить на карте:

- ✅ Кабинеты (с номерами)
- ✅ Коридоры
- ✅ Лестницы
- ✅ Туалеты
- ✅ Выходы
- ✅ Лаборатории
- ✅ Актовый зал
- ✅ Столовая
- ✅ Административные кабинеты

## Цветовая схема (рекомендация):

```
Учебные кабинеты: #3498db (синий)
Лаборатории: #9b59b6 (фиолетовый)
Административные: #e67e22 (оранжевый)
Коридоры: #ecf0f1 (светло-серый)
Лестницы: #e74c3c (красный)
Туалеты: #1abc9c (бирюзовый)
Столовая: #f39c12 (желтый)
```

## После создания SVG:

1. Поместите файлы в `data/maps/`
2. Обновите компонент Map.vue для отображения SVG
3. Добавьте интерактивность (клик по кабинету)

## Нужна помощь?

Если у вас есть фотографии планов этажей или схемы ККРИТ, я могу помочь создать SVG на их основе. Просто скиньте изображения!

## Альтернатива: Использовать готовый план

Если у ККРИТ есть официальные планы эвакуации или схемы здания, можно:
1. Сфотографировать их
2. Обвести в Figma/Inkscape
3. Экспортировать в SVG
