## 🚀 Инструкция по запуску

### Шаг 1: Запуск Backend

```bash
cd backend
python run.py
```

**Ожидаемый результат:**
```
INFO:     Started server process
INFO:     Waiting for application startup.
INFO:     Application startup complete.
INFO:     Uvicorn running on http://127.0.0.1:8000
```

**Проверка:**
- Откройте http://localhost:8000 - должен вернуть JSON с информацией об API
- Откройте http://localhost:8000/docs - Swagger документация
- В логах должно появиться: "Запуск фоновой задачи обновления новостей"

### Шаг 2: Запуск Frontend

```bash
cd frontend
npm run dev
```

**Ожидаемый результат:**
```
VITE v5.x.x  ready in xxx ms

➜  Local:   http://localhost:5173/
➜  Network: use --host to expose
```

**Проверка:**
- Откройте http://localhost:5173
- Должна загрузиться главная страница
- Проверьте страницу "Сотрудники" - должны быть аватарки
- Оставьте страницу на 3 минуты без действий - должен произойти возврат на главную

## 🧪 Тестирование функционала

### 1. Проверка API

```bash
# Получить всех сотрудников
curl http://localhost:8000/api/v1/staff/

# Получить расписание группы
curl http://localhost:8000/api/v1/schedule/group/9ИСП-1.25

# Получить список аудиторий
curl http://localhost:8000/api/v1/rooms/

# Получить новости
curl http://localhost:8000/api/v1/news/
```

### 2. Проверка аватарок сотрудников

1. Откройте http://localhost:5173/staff
2. Убедитесь, что у каждого сотрудника есть круглая аватарка
3. Наведите курсор - аватарка должна увеличиться
4. Проверьте, что цвета фона разные

### 3. Проверка таймера бездействия

1. Перейдите на страницу "Расписание"
2. Не трогайте мышь/клавиатуру 3 минуты
3. Приложение должно вернуться на главную
4. В консоли браузера (F12) должно появиться: "Таймер бездействия: возврат на главную"

### 4. Проверка фоновых задач

1. Запустите backend
2. Подождите 30 секунд
3. В логах должно появиться: "Запуск обновления новостей..."
4. Проверьте таблицу news - должны появиться новые записи (если сайт доступен)

## 🔧 Устранение неполадок

### Backend не запускается
```bash
# Проверьте зависимости
cd backend
pip install -r requirements.txt

# Проверьте базу данных
python check_db.py
```

### Frontend не собирается
```bash
# Переустановите зависимости
cd frontend
rm -rf node_modules package-lock.json
npm install
```

### Таблица rooms пустая
```bash
# Запустите скрипт заполнения
python populate_rooms.py
```

### Новости не обновляются
- Проверьте доступность сайта kraskrit.ru
- Проверьте логи backend на наличие ошибок
- Можно обновить вручную: POST /api/v1/news/refresh
