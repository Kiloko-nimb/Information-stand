# ✅ Парсинг новостей - ГОТОВО!

## Что сделано:

1. ✅ Таблица `news` создана в базе данных
2. ✅ Добавлены 3 тестовые новости
3. ✅ Backend запущен на http://localhost:8000
4. ✅ API endpoint `/api/v1/news` работает

## Как проверить:

### 1. Backend уже запущен
Backend работает на порту 8000.

### 2. Проверить API в браузере:
Открой в браузере: http://localhost:8000/api/v1/news

Должен вернуться JSON с новостями.

### 3. Запустить Frontend:
```bash
cd frontend
npm run dev
```

Открой http://localhost:5173 - новости должны загрузиться автоматически!

## Что происходит:

1. **При загрузке страницы** - frontend делает запрос к `/api/v1/news`
2. **Backend возвращает** 3 новости из базы данных
3. **Frontend отображает** их в блоке "Новости колледжа"
4. **Кэширование** - новости сохраняются в localStorage для offline-режима

## Если новости не отображаются:

1. Открой DevTools (F12) в браузере
2. Перейди на вкладку Console
3. Посмотри на ошибки (если есть)
4. Перейди на вкладку Network
5. Найди запрос к `/api/v1/news` - проверь статус (должен быть 200)

## Автоматическое обновление:

Backend автоматически обновляет новости с сайта kraskrit.ru каждые 6 часов.

**ВАЖНО:** Селекторы HTML в `backend/app/services/news_parser.py` нужно адаптировать под реальную структуру сайта колледжа!

## Принудительное обновление:

```bash
curl -X POST http://localhost:8000/api/v1/news/refresh
```

Это запустит парсинг новостей с сайта прямо сейчас.
