# Импорт расписания из Excel

## Использование

Для импорта расписания из Excel файла в базу данных:

```bash
cd backend
python -m app.utils.import_schedule "путь/к/файлу.xlsx" "YYYY-MM-DD"
```

### Пример

```bash
python -m app.utils.import_schedule "../13.04.2026.xlsx" "2026-04-13"
```

## Формат Excel файла

Файл должен содержать:
- Столбец 1: Номер пары (0, 1, 2, 3, 4, 5, 6, 7)
- Столбец 2: Время занятий (например, "8:10-8:55")
- Далее пары столбцов для каждой группы:
  - Нечетный столбец: Название предмета и ФИО преподавателя
  - Четный столбец: Номер аудитории

## Структура данных

Скрипт автоматически:
- Извлекает названия групп из заголовков
- Парсит время начала и окончания занятий
- Извлекает ФИО преподавателя из названия предмета
- Сохраняет все данные в таблицу `schedules`

## Отображение на сайте

После импорта расписание доступно через:
- Поиск по группе: `/schedule/group/{название_группы}`
- Поиск по преподавателю: `/schedule/teacher/{ФИО}`
- Поиск по кабинету: `/schedule/room/{номер}`

Интерфейс автоматически:
- Сортирует пары по порядку
- Подсвечивает текущую пару
- Отображает дату и день недели
- Адаптируется под мобильные устройства
