# Exploratory scripts

Отладочные скрипты, которые использовались при первичном разборе формата
исходных файлов. Они **не** грузят данные в БД, а только печатают структуру
таблиц. Рабочие лоадеры — в корне репозитория:

- `load_staff.py` — загрузка педсостава из DOCX.
- `load_schedule.py` — загрузка расписания из PDF.
- `backend/app/utils/import_schedule.py::import_schedule_from_excel` —
  загрузка расписания из Excel (`.xlsx`).
